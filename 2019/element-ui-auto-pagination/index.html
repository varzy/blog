使用 ElementUI 构建一个自动分页组件

Aiden's Blog
2019-01-14
Code
frontend, original, tutorial, typescript, vue

在 ElementUI 中提供了 el-table 和 el-pagination 两个组件，但这两个组件间没有任何关联，我们必须自行实现分页功能。为了能尽可能的复用分页代码，我编写了一个可以自动实现表格数据分页的包裹组件。

使用也非常的简单，只需要将全量数据通过 prop 传入组件，并且通过 slot 插入表格内容即可。

这个组件目前只适用于静态数据的分页，理论上使其支持动态分页也并非难事。我们可以将接口地址传入组件内部，每当页码改变时就请求接口获取数据，并实时插入到 totalData 数组中。
