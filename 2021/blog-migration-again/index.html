停更警告：博客已迁移至 blog.varzy.me

Aiden's Blog
2021-03-15
Essay
blog, original, recommend

没错，我又要更换博客平台了。如第一篇所写的一样，我的人生确实少不了折腾。

最近几年我迷上了笔记软件 Notion，我已经把技术类文章全面迁移到了 Notion，而这里就如同副标题一般，仅保留了一些思索与臆语。我钟情于 Notion 的编辑体验，也早就想把所有文章全部迁移到 Notion，但可惜 Notion 的 SEO 优化实在是个问题。但随着近期 Notion API 的成熟，我觉得是时候尝试迁移到 Notion 了。

写下这段文字时，我用上了 Nobelium 这套解决方案，本质上是使用 Nextjs + Notion API 进行服务端渲染，配合 Vercel 可以完美实现 Notion 内编写实时生效。

我相信 Nobelium 仍不是完美的解决方案，比如现在想要更换主题需要直接改动源码，这对于非前端程序员来说不甚友好。但 Nobelium 无疑提供了一种思路。我相信未来会有更好得轮子，我会自己造一个轮子也说不定。好了就这样吧。这里的寥寥数篇文章是托管在 GitHub Pages 上的，如果没什么意外我也不会特意停掉，而新的博客地址是:

https://blog.varzy.me

Enjoy and Bye~
